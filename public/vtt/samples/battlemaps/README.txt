Scryer sample battlemaps (shipped with the app)

Add PNG / WebP / SVG files here, then register each one in:
  lib/vtt/sample-battlemaps.ts

Use stable ids (e.g. scryer-forest-road) and set publicPath to:
  /vtt/samples/battlemaps/your-file.webp

Grid: pixelSize should match how many pixels one square is on the image.

These appear in the VTT → Assets panel → Samples → Maps.
Loading a sample creates a campaign media_items row when needed.
